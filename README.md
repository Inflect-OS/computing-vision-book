# Computing Vision

Our current computing paradigm is to integrated in our way of life that we take it for granted. We buy computers or smartphones, we install applications, store files and
use one particular application, the web browser, to access more apps, to access content and to communicate with the world around us or the whole planet. Networks are faster
than ever, computers are more powerful than ever, with new specialized chips for gaming and running AI workloads without a sweat.
